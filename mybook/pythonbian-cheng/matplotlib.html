1. Matplotlib

Matplotlib的方便之处在于使用Jupyter notebook时能实时画图并显示，这样一来，远程即可看到服务器上的图。

1.1. 柱状图 bar

plt.bar(range(len(mean_hist)), mean_hist, width=1.0, linewidth=1)
plt.xlim([0,256])
#设置横坐标的文字说明
plt.xlabel('Grade')
#设置纵坐标的文字说明
plt.ylabel('Frequency')
#设置标题
plt.title('Title')
plt.show()

1.2. 散点图 plot

plot(X(pos, 1), X(pos, 2), 'k+', 'LineWidth', 2, 'MarkerSize')

plot(X(pos, 1), X(pos, 2), 'k+', 'LineWidth', 2, 'MarkerSize', 7);

plot(X(neg, 1), X(neg, 2), 'ko', 'MarkerFaceColor', 'y', 'MarkerSize', 7);

1.3. 在图上画框 Rectangle

import matplotlib.pyplot as plt
import matplotlib.patches as patches

image = cv2.imread(png_dir + image_name + '.png')
fig1 = plt.figure()
ax1 = fig1.add_subplot(111, aspect='equal')
ax1.imshow(image)
ax1.add_patch(
    patches.Rectangle(
        (bbox[0], bbox[1]),   # (x,y)
        bbox[2],          # width
        bbox[3],          # height
    )
)
plt.imshow(image)
